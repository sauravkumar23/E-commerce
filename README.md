# E-website product managemant portal

> This platform built using React JS ,Node Js ,Express Js,MongoDB.

## Features

- Register and Login

* User

  - Add a Product
  - Update a Product
  - Change a Product
  - Search a Product


### Install Dependencies (frontend & backend)

```
cd front-end
npm install
cd backend
npm install
```

### Run

```
# in one terminal
npm  start (For frontend)
# in other terminal 
npm start (For backend)
